    Otimização de Marketing e Captação de Clientes para Seguros

    Contexto: Uma empresa de seguros de saúde deseja identificar quais clientes podem estar interessados em um novo produto: um seguro de automóvel. Como a equipe de vendas possui um limite de 20.000 ligações, é essencial priorizar os clientes mais propensos a comprar para otimizar os recursos e maximizar as conversões.

    Resultado Final: O projeto resultou em um modelo preditivo alocado em uma API na nuvem, que calcula a probabilidade de compra de cada cliente. Os resultados são integrados diretamente ao Google Sheets, permitindo que o time de vendas acesse e utilize os dados de forma prática e em tempo real. Com base nessas probabilidades, a equipe pode priorizar suas abordagens, concentrando esforços nos clientes com maior chance de conversão.

    Ferramentas e Tecnologias Utilizadas:

    • Python (Pandas, Scikit-learn, Matplotlib, Seaborn, Flask)
    • Visual Studio Code
    • Jupyter Notebook
    • Git e GitHub
    • Algoritmos de Machine Learning: Random Forest Classifier, Regressão Logística
    • Avaliação de Modelos: Precision top k, Recall top k, Curva de Ganho

    Previsão de Vendas Rossmann

    Contexto: Os gerentes da Rossmann enfrentam o desafio de prever com precisão as vendas diárias com até seis semanas de antecedência, influenciadas por fatores como promoções, concorrência, feriados escolares e estaduais, sazonalidade e características locais.

    Resultado Final: Uma solução utilizando um algoritmo que analisa esses fatores e prevê as vendas com até 6 semanas de antecedência. O modelo foi integrado a um bot no Telegram, proporcionando uma interface prática para os gerentes acessarem as previsões.
